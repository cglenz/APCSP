

# Blinky Light Tutorial
	Exactly what it sounds like - makes an LED connected to the arduino blink - oh yeah!

## Materials
	- Arduino
	- LED
	- 220 Ohm Resistor
	- Wires
	- Breadboard

## Sketch

	![blinky_light_sketch](blink_led.png)

## Code

	```
	void setup() {
		pinMode(LED_pin, OUTPUT);
	}

	void loop() {
		digitalWrite();

	}

	```
